17. Imprimir desde Emacs

Pág.Anterior  |  Índice |  Pág.Siguente

Vamos a dar unas simples "recetas" para imprimir desde Emacs:

Imprimir un buffer con numeración de página y cabeceras:

  • Opción de menú : File --> Print Buffer
  • Comando "largo" : M-x  print-buffer <Enter>

Imprimir directamente un archivo, sin numeración de página ni cabeceras:

  • Comando "largo" : M-x  lpr-buffer <Enter>

Impresión de regiones seleccionadas, con numeración de páginas y cabeceras:

  • Opción de menú : File --> Print Region
  • Comando "largo" : M-x  print-region <Enter>

Impresión de regiones seleccionadas, sin numeración de páginas ni cabeceras:

  • Comando "largo" : M-x  lpr-region <Enter>

Impresión de un archivo desde Dired:

Para imprimir un archivo desde  Dired,  situar el cursor en su linea y teclear  P.  o bién ejecutar la opción de menú : Operate --> Print

Emacs nos mostrará, en el mini-buffer, el comando de impresión por defecto, que podrá ser editado y modificado.


Impresión de un archivo como Post-Script:

  • Opción de menú : File -->Postscript Print Buffer
  • Comando "largo" : M-x  ps-print-buffer-with-faces <Enter>
  • Imprime el buffer con los atributos de texto.
  • Opción de menú : File -->Postscript Print Region
  • Comando "largo" : M-x  ps-print-region-with-faces <Enter>
  • Imprime la región seleccionada con los atributos de texto.
